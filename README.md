# C 盘清理

基于 disk-cleaner 思路的 C 盘分析与小工具：按类别预览可清理文件，支持多选删除与在资源管理器中打开路径。

## 功能

- **扫描 C 盘**：扫描临时文件、缓存、日志等目录，与 disk-cleaner 技能逻辑一致
- **按类别展示**：临时文件 / 缓存 / 日志，每类内按文件大小从大到小排列
- **多选**：类别前有复选框可全选该类别，每行前有复选框可单独勾选
- **路径跳转**：点击文件路径可在资源管理器中打开并选中该文件
- **删除选中**：仅允许删除扫描范围内的文件，受保护类型（如 .exe）不会出现在列表中

## 运行

```bash
# 在项目根目录 d:\clean_disk
pip install -r requirements.txt
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

浏览器访问：http://localhost:8000/

## 结构

- `backend/scanner.py`：C 盘扫描逻辑（临时/缓存/日志路径与安全过滤）
- `backend/main.py`：FastAPI 接口（扫描、打开路径、删除）
- `frontend/index.html`：单页界面

## 说明

- 删除前会二次确认；仅会删除扫描范围内、非受保护类型的文件
- 首次使用建议先点「扫描 C 盘」查看列表，再勾选需要清理的项后点「删除选中」
